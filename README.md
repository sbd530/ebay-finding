# :wrench: Installation

- Install Node.js LTS version.
  - https://nodejs.org/ko/download/
  - Check version by `node -v` and `npm -v`

- Git clone or download code.

```bash
$ npm install
```

# :computer: Execution

- Create `.env` file at the root directory to execute app.
  - Refer to `.env.sample` file.

```
.
├── .env.sample
├── .env
...
```

- Run the app.

```bash
$ npm run start
```
